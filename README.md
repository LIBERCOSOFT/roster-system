## ğŸ“– About the Project <a name="about-project"></a>

**Roster Planner** is a comprehensive staff scheduling application designed to streamline workforce management. It provides real-time roster visibility with a live view mode and advanced planning capabilities through the planner mode. Users can drag and drop staff members onto the calendar to schedule shifts, switch between different date views, and access detailed event information at a glance.

## ğŸ’» Getting Started <a name="getting-started"></a>

### Prerequisites <a name="prerequisites"></a>

- **Node.js** v18 or later
- **pnpm** v8 or later (or npm/yarn as alternatives)

### Setup <a name="setup"></a>

Clone the repository:

```sh
git clone https://github.com/LIBERCOSOFT/roster-system.git
cd roster-system
```

### Installation <a name="installation"></a>

Install dependencies using pnpm:

```sh
pnpm install
```

Or using npm:

```sh
npm install
```

### Running the Application <a name="running"></a>

Start the development server:

```sh
pnpm dev
```

Or using npm:

```sh
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).
